How It Works

Searching for Transactions

  • Meter Number: Enter the meter number for which you want to retrieve transactions.
  • Period: Select the "Start Date" and "End Date" to define the transaction period.
  • Search Button: Click the "Search" button. The system will then retrieve the transaction data for the specified meter number and period.

Viewing Details

  • View Details: Click the "View Details" button in the table to see more information about a transaction.
  • Scroll: Use the scroll bar to view all results if there are many transactions.

General Tips

  • Accurate Information: Ensure the meter number and date range are correct to get the desired transactions.
  • Session Security: Your session is secure. Always remember to log out when you are finished using the portal.
  • No Results: If no data is found for the criteria, a message will appear in the table area.
